iOS多国化学习笔记

发表于 2016-11-22

xib和storyboard文件

  1. Project -> Localizations中选择需要本地化的语言
  2. xib文件选择Localization中的Base，并勾选需要本地化的语言
  3. storyboard文件中Localization中，勾选Base（默认已经勾选）和需要本地化的语言
  4. 在相应的XXX.lproj目录中，找到XXX.strings文件，修改需要本地化的内容

代码中的文字

  1. 找到需要本地化的内容，替换成NSLocalizedString(key, comment: comment)
  2. 使用命令genstrings *.swift -o en.lproj行生成Localizabel.strings文件
  3. 添加生成的strings文件到工具目录，并且勾选和需要本地化的语言
  4. 修改需要本地化的内容

InfoPlist.strings

可以修改plist中显示的内容，例如App的名字
